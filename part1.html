Question 1: Converting Between Number Bases (6 marks):

Consider A is the last 3 digits of your AUT ID number, so if your ID is 123456 then A = 456.

My Id: 20104599

Perform the following conversions between different number based systems (3 marks for each):
Assume the number A is a decimal number (base 10):
A = 599

1. Convert 599 (from Decimal) to Binary

599 / 2 = 299 r 1
299 / 2 = 149 r 1
149 / 2 = 74 r 1
74 / 2 = 37 r 0
37 / 2 = 18 r 1
18 / 2 = 9 r 0
9 / 2 = 4 r 1
4 / 2 = 2 r 0
2 / 2 = 1 r 0
1 / 2 = 0 r 1

Therefore 59910 in base 2 = 10010101112

2. Convert 599 (from Decimal) to Hexadecimal

599 / 16 = 37 r 7
37 / 16 = 2 r 5
2 / 16 = 0 r 2

Therefore 59910 in base 16 = 25716
